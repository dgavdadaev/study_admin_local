В качестве базового диапазона используется: **10.128.0.0/16** - Сегментация реализована посредством технологии VLAN - Межсетевой обмен осуществляется через центральный шлюз - Пользовательские группы разделены по функциональному назначению ## Физический уровень - Основной маршрутизирующий узел: `msk-donskaya-ngaforov-gw-1` - Коммутаторы уровней доступа и распределения - Выделенный серверный сегмент - Отдельные пользовательские зоны подключения ## Схема L1 ![Схема L1 — физическая топология](01.png){ width=80% } ## Логическая сегментация (VLAN) В сети применяются следующие VLAN: - VLAN 2 — management - VLAN 3 — servers - VLAN 101 — dk - VLAN 102 — departments - VLAN 103 — adm - VLAN 104 — other ## Схема L2 ![Схема L2 — распределение VLAN](02.png){ width=80% } ## IP-адресация 10.128.0.0/16 Адресное пространство разделено на подсети с префиксом /24: - 10.128.0.0 — серверная ферма - 10.128.1.0 — управление - 10.128.2.0 — point-to-point - 10.128.3.0 — ДК - 10.128.4.0 — кафедры - 10.128.5.0 — администрация - 10.128.6.0 — другие пользователи ## Схема L3 ![Схема L3 — логическая адресация](03.png){ width=75% } ## Таблица VLAN | VLAN | Назначение | |------|------------| | 2 | Управление | | 3 | Серверная ферма | | 101 | Дисплейные классы | | 102 | Кафедры | | 103 | Администрация | | 104 | Другие пользователи | ## Распределение подсетей /24 | Подсеть | Назначение | |----------|------------| | 10.128.0.0/24 | Серверы | | 10.128.1.0/24 | Управление | | 10.128.2.0/24 | Point-to-Point | | 10.128.3.0/24 | ДК | | 10.128.4.0/24 | Кафедры | | 10.128.5.0/24 | Администрация | | 10.128.6.0/24 | Другие пользователи | ## 172.16.0.0/12 Для альтернативного варианта адресации использованы подсети: - 172.16.0.0/24 — серверы - 172.16.1.0/24 — управление - 172.16.2.0/24 — P2P - 172.16.3.0/24 — ДК - 172.16.4.0/24 — кафедры - 172.16.5.0/24 — администрация - 172.16.6.0/24 — другие пользователи ## 172.16.0.0/12 ![L3 для 172.16.0.0/12](04.png){ width=75% } ## 192.168.0.0/16 Также выполнено планирование адресного пространства для диапазона: - 192.168.0.0/24 — серверы - 192.168.1.0/24 — управление - 192.168.2.0/24 — P2P - 192.168.3.0/24 — ДК - 192.168.4.0/24 — кафедры - 192.168.5.0/24 — администрация - 192.168.6.0/24 — другие пользователи ## 192.168.0.0/16 ![L3 для 192.168.0.0/16](05.png){ width=75% } # Вывод ## Вывод В результате выполнения лабораторной работы: -
