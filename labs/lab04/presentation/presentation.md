Провести подготовительную работу по первоначальной настройке коммутаторов сети и освоить базовые команды конфигурирования сетевого оборудования Cisco.

## Первоначальная настройка коммутаторов

На каждом коммутаторе была выполнена базовая конфигурация.

Основные этапы настройки:
- изменение имени устройства
- настройка интерфейса управления VLAN
- назначение IP-адреса управления
- настройка шлюза по умолчанию
- настройка паролей доступа
- включение шифрования паролей
- создание пользователя администратора
- включение удалённого доступа по SSH
